UI Components

This section provides a collection of UI components that can be used to match the CloudVid UI pattern.

Button

The Button component offers various styles to suit your needs. Be mindful of using the appropriate colors available

            <button class="btn btn-primary">Primary Button</button>
            
            <button class="btn btn-secondary" style="background-color: #d16b42; border: 1px solid #322315;">Secondary Button</button>
            
            <button class="btn custom-button" style="background-color: #322315; color: #fff;">Custom Button</button>

Card

This Card component is perfect for showcasing key features or content.

Feature Title

Feature description goes here.

Learn More 
            <div class="card">
              <img src="images/logo.png" alt="Feature Card Image" class="card-img-top" />
              <div class="card-body">
                <h5 class="card-title">Feature Title</h5>
                <p class="card-text">Feature description goes here.</p>
                <a href="#" class="btn btn-primary">Learn More</a>
              </div>
            </div>

Breadcrumb

A breadcrumb navigation component to indicate the current page's location.

  1. Home
  2. Library
  3. Data
            <ol class="breadcrumb breadcrumb-text-color">
              <li class="breadcrumb-item"><a href="#">Home</a></li>
              <li class="breadcrumb-item"><a href="#">Library</a></li>
              <li class="breadcrumb-item active" aria-current="page">Data</li>
            </ol>

Preloader

A preloader component to indicate that content is loading.

Loading... 
            <div class="spinner-border" style="color: #91C07D;" role="status">
              <span class="visually-hidden">Loading...</span>
            </div>

Progress Bar

A progress bar component to display the progress of a task.

            <div class="progress">
              <div class="progress-bar" role="progressbar" style="width: 70%; background-color: #91C07D;" aria-valuenow="70" aria-valuemin="0" aria-valuemax="100"></div>
            </div>

Alerts

Alert components to provide feedback messages to users.

This is a success alert.
This is a danger alert.
This is a warning alert. 
            <div class="alert alert-success" role="alert" style="background-color: #91C07D;">
              This is a success alert.
            </div>

            <div class="alert alert-danger" role="alert" style="background-color: #d16b42;">
              This is a danger alert.
            </div>

            <div class="alert alert-warning" role="alert" style="background-color: #D8BA8E;">
              This is a warning alert.
            </div>

Dropdown List

Our Dropdown List component allows users to select options from a list. As an input component, it nedds to have rounded borders

Option 1 Option 2 Option 3 
                  <div class="dropdown">
                      <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton" data-bs-toggle="dropdown" aria-haspopup="true" aria-expanded="false" style="background-color: #91C07D; border: 1px solid #322315; color: #fff;">
                          Select an Option
                      </button>
                      <div class="dropdown-menu" aria-labelledby="dropdownMenuButton">
                          <a class="dropdown-item" href="#" style="color: #91C07D;">Option 1</a>
                          <a class="dropdown-item" href="#" style="color: #91C07D;">Option 2</a>
                          <a class="dropdown-item" href="#" style="color: #91C07D;">Option 3</a>
                      </div>
                  </div>

Pad Angle Control

The Pad Angle Control allows you to adjust the pad angle using a slider.

0.25 
              <div class="pad-angle-control">
                <label for="padAngle">Pad Angle:</label>
                <input
                  type="range"
                  id="padAngle"
                  min="0"
                  max="0.5"
                  step="0.01"
                  value="0.25"
                  class="pad-angle-input"
                />
                <span id="padAngleValue">0.25</span>
              </div>
